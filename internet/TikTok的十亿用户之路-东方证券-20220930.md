## TikTok的定义和价值：“有趣”为主的社交媒体平台
## TikTok的信息分发机制：形成的背景及特点
### 信息分类
#### 人的需求角度
+ 有用有关的信息比较刚性，有趣的信息比较弹性（可选消费），目的是获得快乐
+ 有趣信息把时长作为优化目标；有关有用信息平台追求“连接”和“效率”
#### 信息传播最大范围角度
+ “市场规模”的大小：受众范围广泛程度。篮球广，手球窄
+ “市场格局”的集中或分散：单条信息的共鸣范围大小。“中秋假期‘比较集中，”某某地铁站”比较分散“
+ 搞笑剧情、影视娱乐、明星、运动健身、萌宠：市场规模大、单条内容共鸣范围大，数量少，vv大；生活、时尚、美食、评测、颜值：市场规模大、单条内容共鸣范围小（个性化），流量分散，vv少
### 社交媒体信息分发机制
+ 各平台信息分发的差异信息流中呈现什么样的内容、内容的顺序，这取决于平台的信息分发机制，信息分发机制服务于平台的价值定位。信息召回决定内容池，信息排序决定顺序。
+ web2.0 召回范围是好友订阅创作者发布的内容，排序方式是时间，社交属性重
+ 移动互联网时代，用户反馈数据可被收集，信息供给从稀缺走向过剩，推荐算法被广泛运用
### 个性化算法推荐具体指什么
+ 确定平台定位及相应的最终优化目标
+ 拆解到不同的用户互动行为指标
+ 输入数据，预估用户行为，进行推荐
	+ 预测模型：概率
	+ 价值模型：用户价值、作者价值、平台价值、间接价值
+ 长尾内容的覆盖率（”中心化“，”去中心化“），中心化高低无绝对好坏之分，主要适配于信息的类型和用户的需求
+ 好的推荐系统要满足
	+ 对用户，符合用户需求
	+ 对创作者，其生产的内容要被对其感兴趣的用户看到（覆盖率）
	+ 对平台，收到高质量的用户互动反馈数据
### TikTok信息分发的特殊性：最”不社交“的社交媒体平台
+ 依赖用户反馈建立兴趣网络
+ DAU->留存->时长
+ 差异化用户行为：上下划
+ 创作者角度
	+ 流量稳定度低
	+ 中心化程度高于快手，低于bili
+ 激励创作者
	+ 示范效应，流量集中在少数人身上
	+ 不能形成固化的创作者阶层，但创作者要获得稳定的ROI
### TikTok为何能在众多社交媒体巨头中实现用户数和时长的赶超
#### 一定程度上对社交关系、关注关系的”放弃“
+ 社交关系网络
	+ 人的社交范围有限，人对自己的兴趣认知也是有限的
	+ 这类连接建立后，比较稳定，不轻易发生变化
+ TT：利用用户行为之间的相似性对用户进行连接
	+ 建立兴趣网络比建立社交网络的速度更快
	+ 兴趣网络的强”可变性“更加适应人们快速切换的信息消费需求
	+ 兴趣网络的存在也会给用户带来迁移成本：用户对内容的偏好往往是和特定平台的人群及内容池绑定
### ”上下划“在”有趣“的内容消费场景下，是非常高质量的反馈指标
+ 在”上下划“评价体系里，用户看到的所有视频都会得到反馈；点赞评论做不到这一点
+ 由大脑中的”系统1“操控，娱乐场景下，思维系统更多由”系统1“主宰
+ 不需要用户启动”系统2“进行选择，可以打开用户内容消费多样性的上限
## TikTok的产品形态：和信息分发机制相匹配
### 关于TikTok是如何找到目前的产品形态的
+ 多条腿走路，用资源换确定性
	+ 西瓜视频/BuzzVideo 单列连续瀑布流、一屏多条内容，类似youtube
	+ 火山小视频/Vigo Video 双列带封面瀑布流，类似快手
	+ 抖音/TikTok 单屏沉浸式、一屏一条内容，类似musical.ly
### TikTok产品设计特点：”算法友好型“的产品形态
+  产品设计和信息分发的模式应该是相匹配的
+ TikTok的全屏上下划设计是”算法友好型“的
+ TikTok提供的短视频生产工具大幅降低了内容生产的门槛
+ 现有的技术环境下，单屏沉浸式视频信息流+个性化算法推荐分发的模式，可能是当下互联网媒体/社交平台提升用户活跃时长和留存率的最优解
### 关于Musical.ly的产品形态对UGC内容丰富度的意义
+ Musical.ly的产品（短视频+滤镜+模板+BGM）在降低内容生产门槛、丰富UGC内容生态方面具有重要意义
+ 短视频拆解为三个元素
	+ 画面：用”滤镜“优化画面（Ins）
	+ 镜头语言：用”模板“优化镜头切换、表现手法（美拍）
	+ 声音：引入背景音乐表达情绪、增加差异化和趣味性
## 关于内容生态的人工运营、
### 为什么平台的人工运营对于内容生态是必要的
+ 平台的人工运营一定程度可以缓解内容创意的”正外部性“问题
	+ 内容创意（”造梗“）具有一定公共品属性，有很强的正外部性。复制成本基本等于0（非竞争性），可以借鉴和模仿使用（非排他性），对于创意源头的创作者存在激励不足的问题
	+ 平台可选择1.补贴有优质内容创意的创作者，2. 平台自己创造”创意的源头“
+ ”二八定律“的存在，使运营少部分的创作者可以改善大部分用户体验
	+ 少部分的内容贡献大部分的观看量
### TikTok的运营手段
+ 整体上TikTok在运营方面的投入程度比北美互联网公司的投入程度要更高
+ 创作者补贴方面：激励金额高，且有针对垂类创作者的精准激励
+ 在官方组织内容、活动方面：抓住疫情窗口，频次大幅提升
+ 邀请明星入驻方面：已经覆盖大量欧美地区头部明星，配合获客投放的节奏，19、20两年使关键发力时间点
